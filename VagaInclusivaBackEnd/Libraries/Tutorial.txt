1- Entrar no ambiente virtual venv, com o comando: . venv/Scripts/activate

2- Dentro do ambiente virtual, navegar até a pasta Libraries, que está na raiz da pasta VagaInclusivaBackEnd (Dentro da pasta Libraries é onde está o arquivo de instalação do Gdal)

3- Dentro da pasta Libraries, executar o comando: python -m pip install GDAL-3.4.3-cp39-cp39-win_amd64.whl
* Agora o Gdal está instalado no seu ambiente virtual
OBS: Necessario Minimo Python Versão 3.9 para instalar o arquivo. Caso não tenha, é necessario baixar um referente a versão do seu Python no Site: https://www.lfd.uci.edu/~gohlke/pythonlibs/#gdal
OBS 2: Os arquivos Gedal ficam nessa seção do site: GDAL: the Geospatial Data Abstraction Library is a translator library for raster geospatial data formats. <-- Só jogar isso no Control+F.
OBS 3: Para saber a versão python minima no arquivo, é so ver oque vem depois de "cp" no nome do arquivo, então um arquivo que tem "cp311" é para python versão 3.11, "cp37" python 3.7, etc.
OBS 4: Também baixar o arquivo que tenha "win_amd64" no nome, pois é referente a uma versão para windows 64 bits. Exemplo: Se quero um arquivo Gedal para python 3.7 em um windows 64 bits, o arquivo será: GDAL‑3.4.2‑cp37‑cp37m‑win_amd64.whl

4- Pegar todo o Caminho que leva aos arquivos gdal304.dll e geos_c.dll (Os Dois estão em venv/Lib/site-packages/osgeo/) e colocar esse caminho em duas variaveis diferentes, no arquivos settings.py:
GDAL_LIBRARY_PATH = 'Aqui vai o caminho até o arquivo gdal304.dll'
GEOS_LIBRARY_PATH = 'Aqui vai o caminho até o arquivo geos_c.dll'

Exemplo de como vai ficar no arquivo settings.py:
GDAL_LIBRARY_PATH = 'C:/Users/SeuUsuario/Desktop/SuaPasta1/SuaPasta2/VagaInclusivaBackEnd/venv/Lib/site-packages/osgeo/gdal304.dll'
GEOS_LIBRARY_PATH = 'C:/Users/SeuUsuario/Desktop/SuaPasta1/SuaPasta2//VagaInclusivaBackEnd/venv/Lib/site-packages/osgeo/geos_c.dll'
